# Project Explanation

The goal of this web application is to give SKKU students a chance to let them create their own name card. This website will provide three options for design, and each design will ask the user to input the required entries to make the name card. After, the name card will be available for download as a .png file.


# How to install and Run the Project

This program doesn't need to install because this project is web-based project.
So, All you have to do is just going into [this link](https://iamjinhyoungjoo.github.io/skku-namecard-generator/)

### First, click "START DESIGNING". Then you are ready to start designing the namecard
<img width="1512" alt="Screen Shot 2022-05-22 at 5 00 11 PM" src="https://user-images.githubusercontent.com/43237393/169684936-5012d80d-2b89-43cf-a857-709acd7a9736.png">

### Second, choose your theme
<img width="1512" alt="Screen Shot 2022-05-22 at 5 01 28 PM" src="https://user-images.githubusercontent.com/43237393/169684975-c1664ee9-4a25-4b88-8aea-76253c9d26e6.png">
There are three types of theme.
1. Modern
2. Simple
3. Colorful

After deciding what you want, click "Select" to design.

### Thrid, design your namecard
<img width="1512" alt="Screen Shot 2022-05-22 at 5 03 33 PM" src="https://user-images.githubusercontent.com/43237393/169685043-8c6d8bee-5699-48ce-86b9-562b3352f15b.png">

Then, it's time to design your own namecard.

click "이전" to go previous content. 

click "다음" to go next content.

click "Download to .png" if you finished your design.

(See pop-up message will helpful to design your namecard. It will warn you unexpected cases or let you know what content will be inputed currently)

### Check whether the namecard was successfully downloaded.
